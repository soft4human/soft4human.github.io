Unbeatable Football Manager Changelog

1.4.0

  • Added the ability to edit team names
  • Team lineup screen now displays goals scored by each player
  • Players' nationalities now appear in the match report
  • Fan sentiment changes are now shown on the match screen
  • Introduced a team trophy screen
  • New award added: "League Best Scorer" for players
  • Improved design
  • Minor improvements and bug fixes

1.3.0

  • Android 15 support added
  • Match Screen: team vs team statistics added
  • History Screen: filter by tournament added
  • Game balance improved: morale drop of a player increased if his contract ends this season
  • Australia added to the possible nationalities of the players
  • Minor improvements

1.2.1

  • Player Screen: "Salary per Month" row added
  • Player Screen: the contract end date is now highlighted in red if the contract ends this season
  • Match Screen: "first leg result" row added for championship games
  • Scotland and Romania added to the possible nationalities of the players
  • Minor improvements

1.2.0

  • Game balance improved: energy drop for players decreased
  • Bug fixed related to the lack of indication of disabled buttons controls
  • Minor improvements

1.1.3

  • Minor improvements

1.1.2

  • Long press action leading to the Player Screen is now available on the Squad Selection Screen
  • Animation has been added to the buttons
  • Minor improvements

1.1.1

  • Bug fixed related to random app crashes on some devices in case the app was in the background and then reopened again
  • Minor improvements

1.1.0

  • Autosave is implemented
  • Improved Home Screen menu design
  • Implemented redirect to the News Screen on news appearance
  • New design for buttons
  • Minor fixes
